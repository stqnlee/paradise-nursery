# Paradise Nursery - Online Plant Shop (Shopping Cart)

Paradise Nursery is a simple e-commerce shopping cart web application for an online plant shop.

## Features
- Landing page with background image and "Get Started" button
- Product listing grouped into categories
- Add to Cart with button disable after adding
- Dynamic cart icon count in navbar
- Shopping cart page:
  - Increase/decrease quantity
  - Remove items
  - Total per item and total cart amount
  - Checkout button shows "Coming Soon"
  - Continue shopping navigation

## Tech Stack
- React (Vite)
- React Router
- Redux Toolkit
- React Redux
